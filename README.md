# 🎙️ Voice Note Mobile App
# 📖 Введение 
Я решил сделать простой **_блокнот-задачник_**, внедрив в него _голосовые функции_, а также предоставив пользователям бесплатный доступ к _нейросети_, что облегчает жизнь современного человека, где каждая минута на счету.

Также, по-моему мнению, проблемой современного человека является то, что абсолютно все данные проходят через такие крупные корпорации как _Google, Microsoft, Apple_ и др., что лишает человека его права на личную жизнь. 

В моём же приложении есть возможность как пользоваться функциями этих корпорации (в частности, Google), так и ими принебречь (данные отправляются лишь при регистрации аккаунта, а также при использовании _любых_ голосовых функций, подробнее в [политике конфиденциальности](https://github.com/Frellest/voice-note/blob/main/docs/privacy-policy.md.txt))

# О приложении
**Voice note** - современное приложение для создания заметок, задач и напоминаний, главной особенностью которого является голосовой ввод (голосовое управление), а также ИИ-конспектирование содержимого заметки.
# 👥 Аудитория приложения
- **Студенты** - конспект лекции голосом
- **Журналисты** - быстрая фиксация идей
- **Все, кто ценит время** - забудьте о долгом наборе текста, попробуйте новый голосовой ввод на базе _Google распознования речи._

# Ключевые возможности
### 🎤 Голосовой ввод
- Запись заметок голосом через микрофон вашего устройства

# 🛠️ Технологии
Проект построен на современном стеке технологий:

| Компонент | Технология | 🎯 Назначение |
|-----------|------------|---------------|
| **Язык** | Kotlin 1.9+ | 🟣 Основной язык разработки |
| **UI** | Jetpack Compose | 🎨 Декларативный UI от Google |
| **База данных** | Room | 🗄️ Типобезопасная работа с SQLite |
| **Сеть** | OkHttp + Retrofit | 🌐 HTTP-запросы к API |
| **AI** | OpenRouter API | 🤖 Доступ к AI-моделям |
| **Архитектура** | MVVM | 🏗️ Разделение логики и UI |
| **Асинхронность** | Kotlin Coroutines | ⚡ Потоки без блокировок |

- **Jetpack Compose** — декларативный UI от Google, ускоряет разработку
- **Room** — типобезопасная работа с SQLite
- **OpenRouter** — доступ к множеству AI-моделей через единый API

# 🎤 Голосовые команды

Путь к успеху — говорить чётко, внятно, чтобы распознавание речи произошло без существенных искажений.

## 📝 Заметка

### ✨ Создание

**"создай заметку", "добавь заметку", "сделай заметку", "создать заметку"** — далее идет название, можно не называть ничего после этих слов: в таком случае откроется меню *новой пустой заметки*. В случае, если пользователь захотел добавить название, он может называть его после произноса ключевых слов, оглашённых ранее: в таком случае все, что вы скажете далее, будет занесено в заголовок. Также вы можете уточнить, с какого места будет ваше название, добавив ключевое слово **"с названием"**.

---

### 🔍 Открытие

**"открой заметку", "покажи заметку", "найди заметку"** — далее идёт название самой заметки (без ошибок, что важно, учитываются даже знаки препинания).

---

### 🗑️ Удаление

**"удали заметку", "удалить заметку", "удались заметку"** — далее идёт название (как и в прошлом случае, без ошибок). Будьте аккуратны, ибо удалённые заметки восстановить нельзя.

> ⚠️ **Внимание!**  
> **"удали все заметки", "удали всё", "удалить все заметки", "удалить всё", "удались все заметки"**  
> — **КРАЙНЕ ОСТОРОЖНО** — удаляет безвозвратно **все** заметки, без предупреждения!
> 
## ✅ Задачи

### ✨ Создание

**"создай задачу", "добавь задачу", "сделай задачу", "создать задачу"** — можно говорить в произвольной форме, все правила при создании заметки сохраняются, однако вы дополнительно можете сразу задать дату напоминания: советую поэкспериментировать, попробуйте говорить в свободной форме. (может плохо распознаваться, издержки производства 😄)

---

### 🔍 Открытие

**"открой задачу", "покажи задачу", "найди задачу"** — всё так же, как и в открытии заметки.

---

### 🗑️ Удаление

**"удали задачу", "удались задачу"** — всё так же, как и в удалении заметки.

## 🧭 Переход между экранами

| Команда | Куда ведёт |
|---------|-----------|
| **"на главный экран", "к заметкам", "на экран заметок", "главное меню"** | 📋 Главный экран (заметки) |
| **"на экран задач", "к задачам", "к задаче"** | ✅ Список задач |
| **"в меню", "аккаунт", "оформить подписку", "на экран меню"** | 👤 Меню / Аккаунт |
| **"к настройкам", "настройки", "настройка", "на экран настроек"** | ⚙️ Настройки приложения |

💡 **Совет:** Говорите естественно, но чётко. Система понимает фразы в свободной форме — пробуйте разные варианты!
