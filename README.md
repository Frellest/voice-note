# Voice Note Mobile App
# Введение 
Я решил сделать простой **_блокнот-задачник_**, внедрив в него _голосовые функции_, а также доступ к _нейросети_, что облегчает жизнь современного человека, где каждая минута на счету.

Также, по-моему мнению, проблемой современного человека является то, что абсолютно все данные проходят через такие корпорации как _Google, Microsoft, Apple_ и других гигантов, что лишает человека его права на личную жизнь. 

В моём же приложении есть возможность как пользоваться функциями этих корпорации (в частности, Google), так и ими принебречь (данные отправляются лишь при регистрации аккаунта, а также при использовании _любых_ голосовых функций, подробнее в [политике конфиденциальности](https://github.com/Frellest/voice-note/blob/main/docs/privacy-policy.md.txt))

# О приложении
**Voice note** - современное приложение для создания заметок, задач и напоминаний, главной особенностью которого является голосовой ввод (голосовое управление), а также ИИ-конспектирование содержимого заметки.
# Аудитория приложения
- **Студенты** - конспект лекции голосом
- **Журналисты** - быстрая фиксация идей
- **Все, кто ценит время** - забудьте о долгом наборе текста, попробуйте новый голосовой ввод на базе _Google распознования речи._

# Ключевые возможности
### Голосовой ввод
- Запись заметок голосом через микрофон вашего устройства
- Переход на разные экраны с помощью [голосовых команд]()
- Создание, открытие, удаление заметок и задач голосом

### AI-функции
- Создание любых промтов и запросов к бесплатным нейросетям (бесплатная версия)
- Создание конспекта (в случае, если выбран сопутствующий флаг)

### Html-редактор
- Возможность добавлять почти все виды форматирования:
  - Жирный 
  - Курсив
  - Подчёркнутый
  - Зачёркнутый
  - Смена размера текста (заголовок)
  - Заливка
  - Задний фон
  - Маркированный список
  - Нумерованный список
  - Выравнивание по левому/правому краю, а также по центру

# Технологии
Проект построен на современном стеке технологий:

| Компонент | Технология |
|-----------|------------|
| **Язык** | Kotlin 1.9+ |
| **UI** | Jetpack Compose |
| **База данных** | Room |
| **Сеть** | OkHttp + Retrofit |
| **AI** | OpenRouter API |
| **Архитектура** | MVVM |
| **Асинхронность** | Kotlin Coroutines |

- **Jetpack Compose** — декларативный UI от Google, ускоряет разработку
- **Room** — типобезопасная работа с SQLite
- **OpenRouter** — доступ к множеству AI-моделей через единый API

# Голосовые команды
Путь к успеху - говорить чётко, внятно, чтобы распознование речи произошло без существенных искожений

## Заметка

### Создание
**"создай заметку", "добавь заметку", "сделай заметку", "создать заметку"** - далее идет название, можно не называть ничего после этих слов: в таком случае откроется меню _новой пустой заметки_. В случае, если пользователь захотел добавить название, он может называть его после произноса ключевых слов, оглашённых ранее: в таком случае все, что вы скажете далее, будет занесено в заголовок. Также вы можете уточнить, с какого места будет ваше название, добавив ключевое слово "с названием".

### Открытие
**"открой заметку", "покажи заметку", "найди заметку"** - далее идёт название самой заметки (без ошибок, что важно, учитываются даже знаки препинания)

### Удаление
**"удали заметку", "удалить заметку", "удались заметку"** - далее идёт название (как и в прошлом случае, без ошибок). Будьте аккуратны, ибо удалённые заметки восстановить нельзя.
**"удали все заметки", "удали всё", "удалить все заметки", "удалить всё", "удались все заметки"** - **КРАЙНЕ ОСТОРОЖНО** - удаляет безвозвратно все заметки, без предупреждения

## Задачи
### Создание
**"создай задачу", "добавь задачу", "сделай задачу", "создать задачу"** - можно говорить в произвольной форме, все правила при создании заметки сохраняются, однако вы дополнительно можете сразу задать дату напоминания: советую поэксперементировать, попробуйте говорить в свободной форме. (может плохо распозноваться, издержки производства xD)

### Открытие
**"открой задачу", "покажи задачу", "найди задачу"** - всё так же, как и в открытии заметки.

### Удаление
**"удали задачу", "удались задачу"** - всё так же, как и в открытии заметки.
